# Janitri-Login-Automation


This project automates the login page testing of the **Janitri Dashboard** using:

- 🧪 **Java**
- 🧭 **Selenium WebDriver**
- 🧰 **TestNG**
- 🗂️ **Page Object Model (POM)**
- 🧱 **Maven** for dependency management
